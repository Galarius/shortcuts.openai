OpenAI (Images)

Given a prompt, the shortcut uses DALL-E to generate a set of images and asks which ones to save.

Supported platforms:
macOS, iOS, watchOS

Prerequisites:
“Get OpenAI API Key” shortcut.

OpenAI (Completions)

Creates a completion for the provided prompt and parameters using GPT-3, saves the result.

Supported platforms:
macOS, iOS, watchOS

Prerequisites:
“Get OpenAI API Key” shortcut.

OpenAI (Edits)

Given a prompt and an instruction, the shortcut uses text editing API to return an edited version of the prompt.

Supported platforms:
macOS, iOS, watchOS

Prerequisites:
“Get OpenAI API Key” shortcut.

OpenAI (Code Completions)

The shortcut uses Codex model to generate code fragments. It's most capable in Python and proficient in over a dozen languages including JavaScript, Go, Perl, PHP, Ruby, Swift, TypeScript, SQL, and even Shell.

Note: Model code-cushman-001 is configured by default.

Supported platforms:
macOS, iOS, watchOS

Prerequisites:
“Get OpenAI API Key” shortcut.

OpenAI (Chat)

Creates a completion for the provided prompt and parameters using GPT-3. This shortcut tries to imitate ChatGPT by keeping the session context, but is only suitable for short and simple conversations due to the limited amount of tokens (use ChatGPT for long and complex interactions).

Supported platforms:
macOS, iOS, watchOS

Prerequisites:
“Get OpenAI API Key” shortcut.

Get OpenAI API Key

Retrieves the OpenAI API key from Reminders.app.

This helper shortcut is common to all other OpenAI shortcuts. It retrieves your private API key from Reminders.app, making shortcuts exchange relatively secure and allowing you to access the API key on all supported platforms.

Supported platforms:
macOS, iOS, watchOS
